# 📊 Pesquisa de Satisfação - TudoWeb

## 📝 Sobre o Projeto
Este projeto foi desenvolvido para a empresa de marketing **TudoWeb**. O objetivo é coletar e exibir o retorno de uma pesquisa de atendimento ao cliente utilizando a linguagem **Python**.

## 🚀 Funcionalidades
* Coleta de nome, idade e opinião dos entrevistados.
* Utilização da estrutura de repetição `FOR` para gerenciar a quantidade de entrevistas.
* Processamento de dados com estruturas de decisão para classificar as respostas.
* Relatório final com a contagem de respostas "Excelente" e "Ruim".

## 🛠️ Tecnologias Utilizadas
* **Python 3**
* **Git & GitHub**
